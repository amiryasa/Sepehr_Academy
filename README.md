## 🛠️ Installation Steps

1. Clone the repository

```bash
git clone https://github.com/amiryasa/Sepehr_Academy.git
```

2. Change the working directory

```bash
cd Sepehr_Academy
```

3. Install dependencies

```bash
npm install
```

4. Run the app

```bash
npm start
```
